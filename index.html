Usage

It's really simple.
First, include page-notifications.js and page-notifications.css to your webpage.
<
link rel=
"stylesheet"
href=
"page-notifications.css"
>

<
script src=
"page-notifications.js"
>
Then initialize notifications.
var
notifications
= new
PageNotifications();

//or with arguments (see documentation for all)
var
notifications
= new
PageNotifications
({"theme"
:
"dark","parentDiv"
:
"divId"});

And finally you can push notifications.
notifications.
push
({"Title","Content","info",
false
);

Documentation
  • constructor( arguments: JSObject )
    Initializes notifications with given arguments.
    • arguments: { 'theme' : String , 'parentDiv' : String , 'width' : Integer/String }
      JavaScript object of possible arguments
      • theme: "light" / "dark"
      • parentDiv: id of parent element
      • width: notifications width measured in pixels / "fill" to match the parent element width

    Example:
    var
    notifications
    = new
    PageNotifications
    ({"theme"
    :
    "dark","parentDiv"
    :
    "divId"});

  • push( title: String , content: String , type: String , duration: Integer/false )
    Adds a new notification and shows it instantly.
    • title: title of the notification
    • content: content of the notification
    • type: "info" / "success" / "warning" / "error"
    • duration: notification's duration measured in miliseconds / false to disable timer

    Example:
    notifications.
    push
    ({"Some title","Content of the notification","warning",
    3000
    );

  • closeAll( )
    Closes all shown instances of notifications.
